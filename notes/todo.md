📋 KeiAI에 구현해야 할 기능 체크리스트
보고서의 기능 인벤토리에서 우선순위별로 뽑으면:

🔴 Phase 1 — MVP (봇과 대화가 되는 상태)

채팅 처리 파이프라인 (sendChat() 흐름)
프롬프트 엔진 (시스템 프롬프트 + 캐릭터 설명 + 대화 기록 조립)
AI 프로바이더 1개 연동 (OpenAI or Claude)
채팅 UI (메시지 입력 → 스트리밍 응답 표시)

🟡 Phase 2 — 핵심 기능

다중 AI 프로바이더 (프로바이더 추상화 레이어)
로어북 시스템 (키워드 매칭 → 프롬프트 삽입)
캐릭터 카드 가져오기/내보내기 (Tavern PNG 최소한)
토큰 관리 (컨텍스트 리밋 내에서 메시지 자르기)
페르소나 시스템

🟢 Phase 3 — 확장

메모리 시스템 (HypaMemory 급)
플러그인 시스템
파서 & 변수 시스템 ({{variable}})
TTS/이미지 생성 연동

셀프호스팅 지원
